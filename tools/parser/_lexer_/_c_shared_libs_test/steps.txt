# Li bgha y3ref kifch ghadi nkhdem hadchi :

	> 1 - Create src.c (Implementations) / src.h files(Prototypes)
	> 2 - Create main.c that will use these Funcs
	> 3 - use gcc (c compiler) with PIC(pos independent code)
		|
		|_gcc -c -Wall -Werror -fpic src.c  (this will out put src.o file)
	
	> 4 - Gnerate Shared lib from the src.o file
		|
		|_ gcc -shared -o libsrc.so src.o

	> 5 - Link our main.c with the shared library
		|
		|_ gcc -Wall -o output main.c -lsrc
